Quick Start

Three steps to get IronWatch running in your server.

  1. Invite the bot with administrator permissions.
  2. Run /setup to create channels and roles (Tickets, Movement Log, Invoices).
  3. Open /panel to post your ticket & giveaway panels.
# Example permissions needed
- Manage Roles, Manage Channels, Ban Members, Kick Members
- Read Message History, Manage Messages, Use Application Commands
