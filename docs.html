Introduction

IronWatch Security is a next-gen Discord bot for moderation, automation, tickets, invoices, and more.

Setup

1) Invite the bot with admin permissions.
2) Run /setup to auto-create: Tickets, Movement Log, Invoices channels.
3) Use /panel to post a ticket panel and a giveaway panel.

Moderation

/kick @user [reason]
/warn @user [reason]
